# plagin-for-VS-Code
# Этот плагин дает возможность вставлять на место курсора HEX или RGB формат выбранного цвета, цвет можно выбрать в интерфейсе HTML. Чтобы открыть интерфейс, нужно нажать ccntr/cmd + shift+p и ввести insert color. В интерфейсе появиться палитра, на которой можно выбрать желаемый цвет.
